Настройка реальной отправки писем через EmailJS

Шаг 1: Регистрация в EmailJS

1. Перейдите на emailjs.com

2. Создайте бесплатный аккаунт

3. Подтвердите email

Шаг 2: Настройка Email Service

1. В панели EmailJS выберите "Email Services"

2. Добавьте Gmail, Outlook или другой email сервис

3. Скопируйте Service ID

Шаг 3: Создание Email Template

1. Перейдите в "Email Templates"

2. Создайте новый шаблон со следующими переменными:

  • {{from_name}} - имя отправителя
  • {{from_email}} - email отправителя
  • {{phone}} - телефон
  • {{department}} - отдел
  • {{subject}} - тема
  • {{message}} - сообщение

3. Скопируйте Template ID

Шаг 4: Получение Public Key

1. Перейдите в "Account" → "General"

2. Скопируйте Public Key

Шаг 5: Обновление кода

В файле contacts.html замените:

  • YOUR_PUBLIC_KEY на ваш Public Key
  • YOUR_SERVICE_ID на ваш Service ID
  • YOUR_TEMPLATE_ID на ваш Template ID

Альтернативный вариант (без EmailJS)

Если не хотите использовать EmailJS, форма будет:

  • Сохранять данные в localStorage браузера
  • Показывать пользователю номер телефона для связи
  • Вы сможете просматривать сохраненные заявки в консоли браузера
